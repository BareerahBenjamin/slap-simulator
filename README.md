# 👋 打脸模拟器 · 今天想打哪个人格？

一个用来「暴打 MBTI 十六型人格」的解压小网页：选人格、挑工具、**按住脸蓄力**，松手把它抽得 pia~pia~ 转圈圈。带本地记录 +（可选）全网实时排行榜。

- 前端：单个 `slap-simulator.html`，纯静态、零依赖、一个文件即可运行。
- 后端（可选）：两个 Vercel Serverless 函数 + Vercel KV，用来做「全网汇总榜」。不部署也能玩，只是没有全网榜。

---

## 🎮 玩法

1. **选人格**：16 型 MBTI，按四大阵营配色 —— 🟣 分析家 / 🟢 外交家 / 🔵 守护者 / 🟡 探险家。每个人格有专属发型、眉眼、道具和吐槽台词。
2. **挑工具**：11 种手感各异、响声不同的家伙 —— 手、豆橛子、大毛领子、鸡毛掸子、拖鞋、皮条、戒尺、充气锤、咸鱼、枕头、藤条。
3. **蓄力开抽**：按住脸开始蓄力（满蓄力约 2.5 秒），蓄得越满转得越多，**最多 8 圈**；松手时按圈数打出一串「pia pia pia」连击，越猛响声越密。
4. **看它转晕**：转完头上冒金星，人格甩一句专属台词。

顶部三个计数：**已抽（下）**、**累计圈数**、**单次最多圈**。

### 🥚 彩蛋

选 **INTP（逻辑学家）**，任你蓄多满，松手它都会闪身躲开、不计数，还甩你一句挑衅 —— 它永远上不了挨打榜。

## 🏆 排行榜

抽屉贴在屏幕右缘，点把手可收起/展开（状态记进浏览器），里面有两个 tab：

| Tab | 数据来源 | 特性 |
|------|----------|------|
| **我的** | 浏览器 `localStorage` | 离线即时，每台设备各自独立，可「清空我的」 |
| **全网** | Vercel KV（需部署） | 所有玩家汇总，**近实时**（每 4 秒轮询），绿点亮=已连通 |

两个榜都**按人格排名**（累计圈数优先，其次次数），🥇🥈🥉 + 阵营色点，共 16 行。

## ☁️ 部署到 Vercel（含全网榜）

全网榜的凭据**只作为环境变量存在服务端**，浏览器永远拿不到 key。

1. **建 KV**：Vercel 控制台 → 你的项目 → **Storage → Create Database → 选 KV**（Upstash Redis）→ 建好后点 **Connect** 连到本项目。
   - 连上后 Vercel 自动注入 `KV_REST_API_URL`、`KV_REST_API_TOKEN` 等环境变量，`@vercel/kv` 会自动读取，**你不用手动填任何 key**。
2. **部署**：
   ```bash
   vercel --prod
   ```
   首次会自动 `npm install` 装 `@vercel/kv`。
3. 打开线上地址 → 切到「全网」tab，绿点亮即成功；打几巴掌，换个浏览器/手机都能看到同一份汇总。

### 数据流

```
浏览器 ──POST /api/hit  {code, spins}──► Vercel 函数 ──HINCRBY──► Vercel KV
浏览器 ──GET  /api/board（每 4 秒轮询）──► Vercel 函数 ──HGETALL──► Vercel KV
```

`slap:stats` 是一个 Redis hash，字段形如 `INTJ:hits` / `INTJ:spins` / `INTJ:best`。

## 🔒 防刷说明

- `api/hit.js` 校验：`code` 必须是 16 个合法 MBTI 之一，`spins` 为 1~8 的整数（与前端 `MAX_SPINS` 对齐）。
- 写入走 `HINCRBY` 原子自增，`best` 取最大值；同一 IP 每 10 秒最多 30 次。
- 挡得住直接乱写和手抖狂点，**挡不住铁心写脚本刷榜的人** —— 打脸小游戏够用。要更强得加登录 / 验证码，那是另一档工程。

## ⚙️ 可调参数

都在 `slap-simulator.html` 顶部的脚本里：

| 常量 | 含义 | 默认 |
|------|------|------|
| `MAX_CHARGE` | 满蓄力所需毫秒 | `2500` |
| `MAX_SPINS` | 满蓄力最多转几圈 | `8` |
| 轮询间隔 | 全网榜刷新频率 | `4000` ms |

> 注意：改了 `MAX_SPINS` 记得同步 `api/hit.js` 里的 `spins` 上限校验（当前写死 8）。
